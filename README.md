## ğŸš€ Getting Started

### Prerequisites

- Web server (Apache/Nginx) or local development server
- PHP 7.4+ (for contact form functionality)
- Modern web browser

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Hanzalaq24/Gottlich-hardware.git
   cd Gottlich-hardware
   ```

2. **Open with a local server**
   
   Using Python:
   ```bash
   python -m http.server 8000
   ```
   
   Using PHP:
   ```bash
   php -S localhost:8000
   ```
   
   Using Node.js (with http-server):
   ```bash
   npx http-server -p 8000
   ```

3. **Access the website**
   ```
   http://localhost:8000
   ```

### Contact Form Setup

To enable the contact form email functionality:

1. Configure your PHP mail settings in `contact.php`
2. Update the recipient email address (currently set to: gujaratsunmicasurat@gmail.com)
3. Ensure your server has PHP mail() function enabled or configure SMTP
